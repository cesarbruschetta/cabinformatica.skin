cabinformatica.skin Installation
--------------------------------

To install cabinformatica.skin using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``cabinformatica.skin`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        cabinformatica.skin
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
